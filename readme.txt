== Description ==

With this plugin you can easily create a list of domains that are not allowed in your Contact Form 7 forms.

== Installation ==

Extract the files and upload them to your wordpress "/plugins/ directory.

It is important that you have the Contact Form 7 plugin installed on your wordpress.
